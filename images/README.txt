Инструкция по загрузке фоновых изображений для ClubPro
=======================================================

Скачайте следующие изображения и сохраните их в эту папку с указанными именами:

1. hero-bg.jpg
   Ссылка: https://images.unsplash.com/photo-1488646953014-85cb44e25828?w=1920&q=80
   Описание: Туристическое направление (самолет, путешествия)

2. about-bg.jpg
   Ссылка: https://images.unsplash.com/photo-1436491865332-7a61a109cc05?w=1920&q=80
   Описание: Самолет в небе

3. program-bg.jpg
   Ссылка: https://images.unsplash.com/photo-1488646953014-85cb44e25828?w=1920&q=80
   Описание: Туристическая тематика

4. cta-bg.jpg
   Ссылка: https://images.unsplash.com/photo-1469854523086-cc02fe5d8800?w=1920&q=80
   Описание: Путешествие, горы

Как скачать:
1. Откройте каждую ссылку в браузере
2. Нажмите правой кнопкой мыши на изображение
3. Выберите "Сохранить изображение как..."
4. Сохраните с соответствующим именем в папку pro-club\images\

После скачивания всех изображений, страница будет работать с локальными файлами.
